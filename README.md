# 🚀 Ultimate Student Management System

A **modern and professional Student Management System** built using **Spring Boot, Thymeleaf, MySQL/PostgreSQL, and Bootstrap**.
This project allows administrators to **manage students efficiently** with features like adding, viewing, and deleting student records.

---

# 📌 Project Overview

The **Ultimate Student Management System** is designed to simplify student record management.
It provides a clean and responsive interface where administrators can perform essential CRUD operations.

This project demonstrates real-world implementation of:

* Spring Boot MVC Architecture
* Database Integration
* Dynamic UI with Thymeleaf
* Professional Dashboard Design

# ⚙️ Installation Guide

### 1️⃣ Clone the repository

```
git clone https://github.com/vikas00787/ultimate-student-management-system.git
```

### 2️⃣ Open project in IDE

Open the project using:

* Spring Tool Suite
* IntelliJ IDEA
* Eclipse

### 3️⃣ Configure Database

Edit `application.properties`

Example (MySQL)

```
spring.datasource.url=jdbc:mysql://localhost:3306/student_db
spring.datasource.username=root
spring.datasource.password=yourpassword

spring.jpa.hibernate.ddl-auto=update
```

Example (PostgreSQL)

```
spring.datasource.url=jdbc:postgresql://localhost:5432/student_db
spring.datasource.username=postgres
spring.datasource.password=yourpassword
```

---
